# Odoo Cafe POS

A complete Point of Sale (POS) system built for cafes and restaurants. It features a unified dashboard, a live kitchen display system (KDS), customer order tracking, receipt generation, inventory/product management, floor planning, and advanced analytics reporting.

## Features
- **POS Operations**: Unified cart UI for taking orders, applying discounts, and handling payments (cash, card, UPI).
- **Kitchen Display System (KDS)**: Real-time ticket management for chefs to prepare orders dynamically.
- **Reporting & Analytics**: Comprehensive dashboards tracking revenue, active sessions, occupied tables, and top-selling products using interactive charts.
- **Table & Floor Management**: Live overview of restaurant tables, showing occupied vs available statuses.
- **Mobile Orders**: QR code integrations allowing customers to order directly from tables.

## Tech Stack
- **Frontend**: React.js, Vite, React Query, Recharts, CSS Grid/Flexbox
- **Backend**: Node.js, Express.js
- **Database**: MySQL
- **Real-time**: Socket.IO
